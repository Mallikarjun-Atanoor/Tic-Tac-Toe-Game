Tic-Tac-Toe Game (Flask + Docker)

A simple 2-player Tic-Tac-Toe web application built using Python Flask, containerized with Docker and published to Docker Hub.

The application:

Accepts two player names

Provides an interactive 3×3 game board

Detects winner automatically

Maintains a live scoreboard

Persists scores using SQLite

Runs fully inside a Docker container

Uses a Docker volume for persistent storage

Tech Stack

Python 3.12

Flask

SQLite

Docker

Docker Compose (optional)
